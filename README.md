## 自动构建和部署
使用Github Actions + Docker + Docker Compose + Praefik + Nginx实现了项目的自动构建和部署，并可通过域名访问。

## Getting Started

需要先克隆[NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)到本地，然后进入该项目文件夹运行一下命令启动后台服务
```
node app.js
```

安装依赖
```
pnpm install
```
启动开发环境
```
pnpm run dev
```
